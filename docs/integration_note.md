# Integration Map

```text
watermark_anything/  -> model package and extensions
assets/              -> sample images and masks
configs/             -> model configs
checkpoints/         -> params and local checkpoints
notebooks/           -> inference utilities
train.py             -> training entrypoint
requirements.txt     -> dependency list
```

Extension modules:

```text
watermark_anything/extensions/
```

The unified runner uses the repository root as `ProjectRoot`.
