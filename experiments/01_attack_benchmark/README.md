# 01 攻击基准评测

## 目标

本实验建立鲁棒水印的攻击评测基线。基础复现只能说明水印在干净图像上能否被提取，不能说明它在真实传播中是否可靠；攻击基准评测则把压缩、缩放、裁剪、局部遮挡和图像扰动纳入同一套统计流程。

## 创新设计

本模块的创新点不是只看单一攻击，而是把多种破坏方式整理成统一评测管线。每次实验都会先生成水印图，再对水印图施加不同强度的攻击，最后回到同一个提取接口统计恢复结果。

这样的设计有三个好处：

- 能发现水印最脆弱的攻击类型，而不是只报告平均效果。
- 能把后续改进统一放到同一组攻击上比较，避免实验口径变化。
- 能同时观察画质指标和消息恢复指标，判断提升是否以明显视觉损伤为代价。

## 评测内容

实验覆盖的攻击包括：

- JPEG 压缩：模拟社交平台和文件转发中的有损压缩。
- 缩放：模拟缩略图、预览图和再次导出。
- 裁剪：模拟局部截图和画面重构。
- 局部移除：模拟部分区域被覆盖、抹除或替换。
- 常见图像扰动：用于检查模型对轻微处理的稳定性。

## 结果解读

重点看不同攻击下的消息恢复率是否下降，以及下降是否集中在某些攻击类型上。如果某类攻击明显更难恢复，它就会成为后续空间冗余、强度配置或区域同步模块优先处理的方向。
