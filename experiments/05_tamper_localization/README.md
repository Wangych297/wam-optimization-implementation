# 05 篡改定位

## 目标

本实验把水印提取从“整张图是否通过验证”扩展到“哪一块区域可能被破坏”。对于信息安全场景来说，完整性保护不只需要知道图片被改过，还需要尽量指出可疑位置，方便后续人工核查或证据保全。

## 创新设计

篡改定位的核心做法是把图像划分成局部区域，分别进行水印提取和置信度统计。如果某些区域的提取结果明显低于其他区域，就可以把这些区域标记为疑似篡改位置。

这相当于把水印从单一认证标记扩展为局部完整性探针：

- 正常区域应当能提取出稳定消息或保持较高置信度。
- 被覆盖、抹除或替换的区域会出现恢复率下降。
- 多区域统计可以形成热力图或区域级判断，而不是只给出一个全局成败结果。

## 工程实现

`localizer.py` 会构造局部破坏样本，并对不同区域的提取结果进行统计。输出结果用于比较预测篡改区域和实际破坏区域之间的重合程度。

## 结果解读

重点关注两类指标：

- 定位能力：疑似区域是否能覆盖真实被破坏区域。
- 误报情况：未被破坏的区域是否被错误标为异常。

如果定位能力较好但误报较高，说明需要改进阈值或区域划分；如果误报低但漏检严重，说明水印覆盖和提取策略还不够密集。
